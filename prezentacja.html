Problem

  • Wyszukiwarka — znajdziesz plik w sekundy

03
Dla kogo?

CloudDrive jest dla każdego, kto ma pliki w sieci

🎓

Student

Notatki, projekty, prezentacje — wszystko wgrywasz raz i masz dostęp z dowolnego miejsca: laptopa w domu, telefonu w komunikacji, komputera w bibliotece. Koniec z pendrivami.

💼

Pracownik

Raporty, dokumenty firmowe, pliki projektów — wgrywasz szybko, wyszukujesz błyskawicznie. Nie musisz pamiętać, w którym folderze co leży. Bezpieczne przechowywanie bez kompromisów.

🎨

Hobbysta / twórca

Zdjęcia, muzyka, projekty graficzne — przechowuj i organizuj swoje prace twórcze we własnej strukturze folderów. Wszystko posortowane tak, jak lubisz, dostępne natychmiast.

Słowem — dla każdego, kto ceni porządek i chce mieć swoje pliki zawsze pod ręką.

04
Funkcjonalności

Co potrafi CloudDrive?

⬆️

Przesyłanie plików

Kliknij przycisk lub po prostu przeciągnij pliki z komputera i upuść je w aplikacji. Możesz wgrać wiele plików jednocześnie — bez ograniczeń.

⬇️

Pobieranie

Każdy plik pobierasz jednym kliknięciem. Chcesz pobrać cały folder? Aplikacja automatycznie spakuje go do pliku ZIP i wyśle jako całość.

📁

Zarządzanie folderami

Twórz nowe foldery, usuwaj niepotrzebne, zmieniaj nazwy jednym kliknięciem. Nawigacja breadcrumb pokazuje, gdzie aktualnie jesteś w strukturze katalogów.

🔍

Wyszukiwanie

Wpisz nazwę pliku — wyniki pojawiają się natychmiast. Działa błyskawicznie nawet wtedy, gdy masz setki plików w różnych folderach.

⊞

Widok listy i siatki

Przełączaj między widokiem listy a widokiem siatki jednym kliknięciem. Wybierz ten, który bardziej Ci odpowiada — aplikacja zapamiętuje Twój wybór.

🔐

Bezpieczne logowanie

Dostęp do aplikacji wymaga logowania. Każde połączenie zabezpieczone jest tokenem JWT — Twoje pliki są widoczne wyłącznie dla Ciebie.

07
Jak to działa?

Cztery proste kroki

1

Zaloguj się

Wpisujesz e-mail i hasło. Konto jest zabezpieczone tokenem JWT — tylko Ty widzisz swoje pliki. Żadnych zewnętrznych kont, żadnych powiązanych usług.

2

Przeglądaj

Widzisz wszystkie swoje pliki i foldery od razu. Sortuj po nazwie lub rozmiarze, przeszukuj wpisując nazwę, przełącz widok listy lub siatki — tak jak Ci wygodnie.

3

Zarządzaj

Wgrywaj pliki kliknięciem lub drag & drop, pobieraj, zmieniaj nazwy, usuwaj i twórz foldery — wszystkie operacje wykonujesz w kilka sekund bez żadnej instalacji.

4

Dostęp wszędzie

Pliki są bezpiecznie przechowywane w chmurze. Otwórz CloudDrive z komputera, laptopa lub telefonu — z domu, z pracy, z kawiarni, z dowolnego miejsca na świecie.

Żadnej instalacji, żadnych skomplikowanych ustawień — po prostu działa.

10
Fragment kodu

Upload z weryfikacją integralności

Przed wysłaniem pliku liczymy jego sumę SHA-256 — serwer sprawdza, czy nic nie zostało uszkodzone po drodze.

src/api.js JAVASCRIPT 
// Oblicz SHA-256 przed wysłaniem — weryfikacja integralności
async function sha256(file) {
  const buffer     = await file.arrayBuffer();
  const hashBuffer = await crypto.subtle.digest('SHA-256', buffer);
  return Array.from(new Uint8Array(hashBuffer))
    .map(b => b.toString(16).padStart(2, '0'))
    .join('');
}

export async function uploadFile(file, directoryId, token) {
  const checksum = await sha256(file);
  const formData = new FormData();
  formData.append('file',            file);
  formData.append('checksum',        checksum);
  formData.append('idempotencyKey',  crypto.randomUUID());
  if (directoryId) formData.append('directoryId', directoryId);

  const res = await http.post('/v1/files', formData, {
    headers: auth(token)
  });
  return res.data;
}
